Creating an Account

The first step to creating account is visiting github (click github to be redirected)

Click Sign Up in the top right corner, and fill out the information

Once done, you are ready to move on to the next step!


Creating a Repository

A repository is like a folder on your computer. It hosts your code online, and allows you to use 'git' command line commands, which will be used later on in this tutorial

To start, click your account in the top left after signing in, and click "Your Profile"

This is your personal page, where people can visit and view your code. Next, click "Repositories", located near the top of your screen

Click the green button that says New


Now, think of a name for your new repository. This can be anything you wish

However, if you intend to host a site, I would advise making this the same as your github username, and adding '.github.io' to the end of your repository name

For example, if you chose the name 'john-doe', you would name the new repo(repository) john-doe.github.io

Through github, you can host one page per account, and the default url ends in '.github.io'. You can buy a custom URL if you wish, but for people like me who are just having fun coding, it is far from neccessary

Next, you may add a description. Though not neccesary, it can be useful to tell others what the repo is for

Finally, click the green button labeled Create Repository


Your New Repo

If you completed the last step, you should now have a new Repo! Right now, its pretty much empty, unless you added the optional README or a description

For now, you are done here! Next, move to Setup Gitpod, where you will setup an online IDE, or integrated development environment
