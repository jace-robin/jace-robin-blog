Welcome

HTML is the most basic structure, as far as I see it. It stands for "Hypertext Markup Language" It can include many files

For now though, lets focus on the foundation


All elements start with an opening tag. Some examples include div, p, and h1. The format for these tags are '<' and '>'

Some elements require just the opening tag though, like img, which displays images


HTML follows the structure of content placed inside its DOM, or Document Object Model

If you have never seen or heard of anything you've read up to this point, don't get too worried, it's a lot to take in. I'll try my best to take things one at a time

The basic structure of the pages I commonly make look like this:


!DOCTYPE html

html

head

style

style

script

script

head

body

body

html


This code currently displays nothing on the users end, nor does it have any styling. It is more of a template, which you can fill in with anything you choose

The DOCTYPE tag identifies this file as a HTML file. The html tag should always encompass every tag, excluding the DOCTYPE

The head tag contains information that the page uses, but does not display content visible to the user, such as CSS styling and code that can be used for responsive pages. The body tag is where you will be putting what you want the user to see, and what we will focus on next

To start, lets display some basic text, using the <p> element inside our body tags. This is a paragraph, and is often used for basic text. Here, we will use it to display the text "Hello World!"


body

p

Hello World!

p

body


Output:

Hello World!

Note that the fonts I use here are not the default font, so your result will look slightly different
